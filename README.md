## ğŸ”„ Workflow
1. **Setup Environment**: Configure `.env` with database and API keys.
2. **Run Backend**:
   ```bash
   cd server
   npm install
   npm start
   ```
3. **Run Frontend**:
   ```bash
   cd client
   npm install
   npm start
   ```
4. **Access Application**: Visit `http://localhost:3000` to explore the app.
